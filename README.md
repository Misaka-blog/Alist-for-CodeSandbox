# Alist-for-CodeSandbox

在 CodeSandbox 上部署 Alist 文件列表程序

## 项目特点

* 本项目用于在 CodeSandbox 上部署 Alist

## 部署

* 注册 [CodeSandbox](https://codesandbox.io/)
* 上传项目文件至`.codesandbox`文件夹中
* 重启实例，然后在控制台寻找用户名及密码
